# Buyit - in your way
E-commerce website build from scratch with html, css and javaScript for my 2nd year internship project.
This website is fully a frontend project till now and I am thinking to also create a backend like handing the payment and login interface for the user to give them them the personalise experience.
This website works on a single style.css, script.js  and auth.js (for login with firebase console) to maintain the cosistency across the different webpages.

<hr>
<h2>Structure of this website</h2>
The main home page of this website is index.html.
This website also contains many sub-pages like:
<hr>
<h3>HTML</h3>
<ul>
  <li>index.html: the main home page of the website.</li>
  <li>product.html: use for listing all the product.</li>
  <li>product-details.html: use for view the details of the webpage, seperated by section and linked the product to view only the details of the specific product.</li>
  <li>cart.html: use for view the cart.</li>
  <li>login.html: use for login (Back-end by firebase console authentication)</li>
  <li>account.html: use for view login account and purchased history</li>
</ul>

<h3>CSS</h3>
<ul>
  <li>style.css: the single css file use for styling whole website to maintain the constaincy across the webpages.</li>
</ul>

<h3>JavaScript</h3>
<ul>
  <li>script.js: use for the function like adding the products to cart and searching products.</li>
  <li>auth.js: use for the authentication (Back-end by firebase console) and handling the user.</li>
</ul>
